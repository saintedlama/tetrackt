# Tetrackt

The terminal music tracker for chiptune and retro-style music with a great typo.

## Waveforms

- Sine
- Square
- Triangle
- Sawtooth
- Reverse Sawtooth
- Noise

## Effects

- Reverb
- Chorus
- Flanger
- Delay
- Distortion
- Bitcrusher
